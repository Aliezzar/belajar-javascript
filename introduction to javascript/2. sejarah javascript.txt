JavaScript awalnya dibuat oleh Brendan Eich dari NetScape dan pertama kali diumumkan dalam siaran pers oleh Netscape pada tahun 1995. JavaScript memiliki sejarah penamaan yang aneh; awalnya diberi nama Mocha oleh pembuatnya, yang kemudian berganti nama menjadi LiveScript. Pada tahun 1996, sekitar setahun kemudian setelah rilis, NetScape memutuskan untuk mengganti namanya menjadi JavaScript dengan harapan memanfaatkan komunitas Java (walaupun JavaScript tidak memiliki hubungan apa pun dengan Java) dan merilis Netscape 2.0 dengan dukungan resmi JavaScript.
